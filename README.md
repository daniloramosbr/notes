# Bloco de Notas

Projeto Full Stack de um Bloco de Notas com várias funcionalidades.


## Funcionalidades

- Responsividade
- Autenticação e Login
- Criação de Notas
- Edição de Notas
- Remocão de Notas

## Stack utilizada

**Front-end:** React, Axios, JavaScript, JWT, Cookies

**Back-end:** Node, MongoDB, Express, Render, BCRYPT
